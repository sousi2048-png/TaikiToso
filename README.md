# 株式会社タイキ塗装 公式ホームページ

株式会社タイキ塗装の公式ホームページです。静岡県東部を中心に外壁塗装・屋根塗装・防水工事を提供する企業のウェブサイトです。

## 📋 プロジェクト概要

このプロジェクトは、株式会社タイキ塗装の企業情報、事業内容、施工事例、お問い合わせ情報を提供する静的ホームページです。

### 主な機能

- **レスポンシブデザイン**: PC・タブレット・スマートフォンに対応
- **スムーズスクロール**: ナビゲーションリンクのスムーズなスクロール機能
- **フェードインアニメーション**: スクロール時の要素のフェードイン効果
- **SEO最適化**: メタタグによる検索エンジン最適化

## 🗂️ プロジェクト構造

```
TakiTosoHP/
├── index.html          # メインのHTMLファイル
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── main.js        # JavaScriptファイル
├── img/               # 画像ファイル用ディレクトリ
└── README.md          # このファイル
```

## 🚀 セットアップ・使用方法

### ローカルでの実行

1. このリポジトリをクローンまたはダウンロードします
2. `index.html` をブラウザで開きます

```bash
# 例: ブラウザで直接開く
open index.html
```

または、ローカルサーバーを起動します：

```bash
# Python 3の場合
python -m http.server 8000

# Node.js (http-server) の場合
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセスします。

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリングとレスポンシブデザイン

## 📝 カスタマイズ方法

`img/` ディレクトリに画像を配置し、`index.html` の該当箇所で参照してください。

### スタイルの変更

### JavaScript機能の追加

`js/main.js` に新しい機能を追加できます。
